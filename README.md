# Decoding Traveler Emotions
**NLP Sentiment Analysis of Travel Blogs — Maria Aziz (2024)**  
ORCID: 0009-0009-2321-4750

> Open research project shared for community feedback (not peer-reviewed).

### Overview
I analyzed travel blogs to understand how travelers express **joy, frustration, and nostalgia** using BERT and LSTM models. The repo includes full training logs, confusion matrices, and visual model comparisons.

### Files in this repo
- `Travel Sentiment NLP Aziz2024.ipynb` – main notebook
- `bert_training_logs.csv` – BERT training metrics
- `bert_confusion_matrix.csv` – evaluation results  
- `figure1_bert_confusion_matrix.png` to `figure4_sentiment_distribution.png` - plots
- `LICENSE` – MIT
